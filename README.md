# Vector-Database
This is a simple vector database implemented in Python. It's structure is relatively easy to understand and it's easy to use. It's also very fast and efficient. It's a good choice for small projects.

The database supports multiple comparison and index/distance algorithms, such as Jaccard, Euclidean and Cosine distance for high-dimensional data queries. It also supports multiple data types, such as strings, integers, floats and booleans.
